# Game_FruitSlot
Primitive Slots Machine Simulator, my very first time exploring gamedev and using GoDot 3.5.
Programed in GDSkript - exported to github as html, so you can try your luck too!

Try it: https://trrrrex.github.io/Game_FruitSlot/ !

_My 7-game learning project — each one a different genre with unique mechanics, all built in GoDot.

Fun fact: I discovered GoDot about 20 hours ago, and this is literally my first code. 😄

The goal? Learn gamedev logic and architecture through these 7 games. Once they're done, I plan to rewrite them in C++ on Unix — that way, I'll already have the problem-solving experience, and can focus on the harder stuff (C++ itself) instead of figuring out how to make a game work._
